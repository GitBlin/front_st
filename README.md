# 沟通培训系统

一个基于HTML5的沟通培训系统，用户通过与NPC角色对话来提升沟通能力。系统提供实时反馈和评估报告。

## 功能特点

### 🎭 多角色对话场景
- **部门经理**: 练习工作汇报和请求支持
- **重要客户**: 提升商务谈判和客户沟通能力
- **技术同事**: 学习跨部门协作和技术交流

### 💬 智能对话系统
- NPC根据用户消息内容和情绪状态智能回复
- 实时显示NPC心情状态（友好/积极/平静/担忧/不满）
- 提供建议回复选项，支持自定义输入

### 📊 能力评估系统
从四个维度评估用户沟通能力：
- **语言表达**: 用词准确性、表达连贯性
- **逻辑思维**: 论述条理性、说服力
- **情感理解**: 对他人情绪的敏感度
- **应变能力**: 根据情况调整沟通策略

### 📈 实时反馈
- 对话过程中提供实时教学反馈
- 显示对话进度和剩余轮数
- NPC状态可视化展示

### 📋 评估报告
- 总分计算和能力维度评分
- 对话统计（轮数、时长）
- 个性化综合评价和改进建议
- 支持报告导出功能

## 使用方法

### 1. 启动系统
直接在浏览器中打开 `index.html` 文件，或使用HTTP服务器运行：

```bash
# 使用Python启动服务器
python -m http.server 8000

# 使用Node.js启动服务器
npx http-server

# 访问 http://localhost:8000
```

### 2. 选择对话场景
- 浏览三个可用的NPC角色
- 点击选择想要练习的场景
- 点击"开始培训"按钮

### 3. 进行对话
- 使用建议回复或输入自定义消息
- 观察NPC状态变化和实时反馈
- 最多可进行15轮对话

### 4. 查看评估报告
- 对话结束后自动生成评估报告
- 查看各维度得分和综合评价
- 可导出报告为文本文件

### 自定义配置
可以通过修改 `script.js` 中的配置来自定义系统：

```javascript
// 调整对话轮数限制
this.maxMessages = 20;

// 修改角色信息
this.characters.newRole = {
    name: '新角色',
    avatar: '🤖',
    // ...
};
```
